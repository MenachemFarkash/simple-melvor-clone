# simple-melvor-clone

A post‑apocalyptic idle / incremental RPG inspired by _Melvor Idle_. The project started as a simple clone, but has grown into its own world with custom mechanics, skills, and a polished user interface.

## 🎮 About the Game

In this game you take on the role of a survivor exploring a devastated world. Instead of traditional fantasy themes, everything is set after a cataclysm—ruined cities, mutated creatures, and scavenged technology. Progression is driven by familiar idle concepts:

- **Resources** are gathered over time or through actions.
- **Skills** train via player input and unlock new content.
- **Combat, crafting, and exploration** are all available with hands‑off progression if you choose.

Despite the familiar backbone of Melvor Idle, the game features brand‑new content and a complete UI overhaul that makes it feel unique.

## ✨ Key Features

- **Post-Apocalyptic Setting** – scavenge, fight mutated beasts, and rebuild from the ashes.
- **Custom Skills** – beyond the standard set, you’ll find new abilities tailored to the world (e.g. `Salvaging`, `Woodcutting`, `Repairing`,`Hunting`,`Researching`).
- **UI Overhaul** – a clean, modern interface designed with accessibility and post‑apocalyptic aesthetics in mind.
- **Modular Codebase** – built in vanilla JavaScript/HTML/CSS, making it easy to extend or fork.
- **Offline & Idle Friendly** – play actively or let the game run in the background; progress continues.
